Winamp2 VQF Decoder Plug-in
Current version : v1.12a

What is this ?

    This nice piece of software (plug-in) makes Winamp, THE standard MP3 player for Windows, like VQF files ... a lot !

Features

    With this plug-in, you can do almost everything you can do with MP3's.
    • Seeking Support works great (unlike the old plug-in(s))
    • You can now tag your vqf's through the plug-in
      • header extention with new tag fields (album, track, genre & date)
      • tag fields with >1024 characters are possible (disabled by default)
    • Kick-ass Streaming support :
      • Seeking for streams
      • Pre-buffering
    • Fully configurable
    • Priority feature
    • Working EQ, DSP & Vis plug-ins

Version history

      • New title formatting logic
